# `seuratToGiotto_OLD`

seuratToGiotto_OLD


## Description

Converts Seurat object into a Giotto object. Deprecated, see [`giottoToSeurat`](#giottotoseurat)


## Usage

```r
seuratToGiotto_OLD(obj_use = NULL, ...)
```


## Arguments

Argument      |Description
------------- |----------------
`obj_use`     |     Seurat object
`...`     |     additional params to pass


## Value

Giotto object
