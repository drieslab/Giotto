# `set_polygon_info`

Set polygon info


## Description

Set giotto polygon spatVector


## Usage

```r
set_polygon_info(gobject, polygon_name = "cell", gpolygon)
```


## Arguments

Argument      |Description
------------- |----------------
`gobject`     |     giotto object
`polygon_name`     |     name of polygons
`gpolygon`     |     giotto polygon


## Value

giotto object
