# `subset_giotto_points_object`

Subset giotto points object


## Description

Subset a single giotto points object


## Usage

```r
subset_giotto_points_object(
  gpoints,
  feat_ids = NULL,
  x_min = NULL,
  x_max = NULL,
  y_min = NULL,
  y_max = NULL
)
```


## Details

Subset on feature ids and on x,y coordinates
