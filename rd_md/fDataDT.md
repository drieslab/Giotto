# `fDataDT`

fDataDT


## Description

show gene metadata


## Usage

```r
fDataDT(gobject, spat_unit = NULL, feat_type = NULL)
```


## Arguments

Argument      |Description
------------- |----------------
`gobject`     |     giotto object
`spat_unit`     |     spatial unit
`feat_type`     |     feature type


## Value

data.table with gene metadata
