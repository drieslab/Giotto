# `create_giotto_points_object`

Create giotto points object


## Description

Create giotto points object


## Usage

```r
create_giotto_points_object(
  feat_type = "rna",
  spatVector = NULL,
  networks = NULL
)
```


## Arguments

Argument      |Description
------------- |----------------
`feat_type`     |     feature type
`spatVector`     |     terra spatVector object containing point data
`networks`     |     feature network object
