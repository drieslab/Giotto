# `create_2d_mesh_grid_line_obj`

create_2d_mesh_grid_line_obj


## Description

create 2d mesh grid line object


## Usage

```r
create_2d_mesh_grid_line_obj(x_min, x_max, y_min, y_max, mesh_grid_n)
```


## Arguments

Argument      |Description
------------- |----------------
`x_min`     |     x_min
`x_max`     |     x_max
`y_min`     |     y_min
`y_max`     |     y_max
`mesh_grid_n`     |     mesh_grid_n
