# `spline_poly`

Convert spline to polygon


## Description

spline polynomial to smooth polygon


## Usage

```r
spline_poly(xy, vertices = 20, k = 3, ...)
```


## Arguments

Argument      |Description
------------- |----------------
`xy`     |     xy
`vertices`     |     vertices
`k`     |     k
`...`     |     additional params to pass
