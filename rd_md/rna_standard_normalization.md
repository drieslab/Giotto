# `rna_standard_normalization`

RNA standard normalization


## Description

standard function for RNA normalization


## Usage

```r
rna_standard_normalization(
  gobject,
  raw_expr,
  feat_type,
  spat_unit,
  library_size_norm = TRUE,
  scalefactor = 6000,
  log_norm = TRUE,
  log_offset = 1,
  logbase = 2,
  scale_feats = TRUE,
  scale_cells = TRUE,
  scale_order = c("first_feats", "first_cells"),
  verbose = TRUE
)
```
