# `giotto_point`

giotto_point


## Description

uses ggplot::geom_point, scattermore::geom_scattermore or scattermore::geom_scattermost


## Usage

```r
giotto_point(plot_method = c("ggplot", "scattermore", "scattermost"), ...)
```


## Arguments

Argument      |Description
------------- |----------------
`list()`     |     geom_point parameters
