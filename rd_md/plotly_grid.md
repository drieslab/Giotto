# `plotly_grid`

plotly_grid


## Description

provide grid segment to draw in plot_ly()


## Usage

```r
plotly_grid(
  spatial_grid,
  x_start = "x_start",
  y_start = "y_start",
  x_end = "x_end",
  y_end = "y_end"
)
```


## Arguments

Argument      |Description
------------- |----------------
`spatial_grid`     |     spatial_grid in giotto object


## Value

edges in spatial grid as data.table()
