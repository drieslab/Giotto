# `reshape_to_mesh_grid_obj`

reshape_to_mesh_grid_obj


## Description

reshape a data point matrix to a mesh grid line object


## Usage

```r
reshape_to_mesh_grid_obj(data_points, mesh_grid_n)
```


## Arguments

Argument      |Description
------------- |----------------
`data_points`     |     data_points
`mesh_grid_n`     |     mesh_grid_n
