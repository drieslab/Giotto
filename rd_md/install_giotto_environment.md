# `install_giotto_environment`

install_giotto_environment


## Description

installation options of giotto environment


## Usage

```r
install_giotto_environment(
  force_environment = FALSE,
  packages_to_install = c("pandas", "networkx", "python-igraph", "leidenalg",
    "python-louvain", "python.app", "scikit-learn"),
  verbose = TRUE
)
```
