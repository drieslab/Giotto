# `plot_feature_raster_density_layer`

plot_feature_raster_density_layer


## Description

low level function to plot density plots at the spatial in situ level


## Usage

```r
plot_feature_raster_density_layer(
  ggobject = NULL,
  spatial_feat_info,
  sel_feat,
  sdimx = "x",
  sdimy = "y",
  alpha = 0.5
)
```


## Details

This function can plot one feature for one modality.


## Value

ggplot
