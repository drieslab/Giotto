# `create_delaunayNetwork2D`

create_delaunayNetwork2D


## Description

Create a spatial 2D Delaunay network.


## Usage

```r
create_delaunayNetwork2D(
  gobject,
  method = c("delaunayn_geometry", "RTriangle", "deldir"),
  spat_unit = NULL,
  spat_loc_name = "raw",
  sdimx = "sdimx",
  sdimy = "sdimy",
  name = "delaunay_network",
  maximum_distance = "auto",
  minimum_k = 0,
  options = "Pp",
  Y = TRUE,
  j = TRUE,
  S = 0,
  verbose = T,
  return_gobject = TRUE,
  ...
)
```
