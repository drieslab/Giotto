# `estimateCellCellDistance`

estimateCellCellDistance


## Description

estimate average distance between neighboring cells


## Usage

```r
estimateCellCellDistance(
  gobject,
  spatial_network_name = "Delaunay_network",
  method = c("mean", "median")
)
```


## Arguments

Argument      |Description
------------- |----------------
`gobject`     |     gobject
`spatial_network_name`     |     spatial_network_name
`method`     |     method
