# `pDataDT`

pDataDT


## Description

show cell metadata


## Usage

```r
pDataDT(gobject, spat_unit = NULL, feat_type = NULL)
```


## Arguments

Argument      |Description
------------- |----------------
`gobject`     |     giotto object
`spat_unit`     |     spatial unit
`feat_type`     |     feature type


## Value

data.table with cell metadata
