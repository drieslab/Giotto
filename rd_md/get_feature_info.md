# `get_feature_info`

Get feature info


## Description

Get giotto points spatVector


## Usage

```r
get_feature_info(gobject, feat_type = NULL)
```


## Arguments

Argument      |Description
------------- |----------------
`gobject`     |     giotto object
`feat_type`     |     name of feature
