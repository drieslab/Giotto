# `set_feature_info`

Set feature info


## Description

Set giotto polygon spatVector for features


## Usage

```r
set_feature_info(gobject, feat_type = NULL, gpolygon)
```


## Arguments

Argument      |Description
------------- |----------------
`gobject`     |     giotto object
`feat_type`     |     name of feat
`gpolygon`     |     giotto polygon


## Value

giotto object
