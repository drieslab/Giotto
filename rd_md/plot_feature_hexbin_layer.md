# `plot_feature_hexbin_layer`

plot_feature_hexbin_layer


## Description

low level function to plot hexbins at the spatial in situ level


## Usage

```r
plot_feature_hexbin_layer(
  ggobject = NULL,
  spatial_feat_info,
  sel_feat,
  sdimx = "x",
  sdimy = "y",
  bins = 10,
  alpha = 0.5
)
```


## Details

This function can plot one feature for one modality.


## Value

ggplot
